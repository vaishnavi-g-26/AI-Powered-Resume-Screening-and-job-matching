AI-Powered Resume Screening and Job Matching System
Project Overview
This project is an AI-based Resume Screening and Job Matching System that automates the process of analyzing resumes and matching them with suitable job roles. It helps recruiters reduce manual effort and improves candidate-job alignment using machine learning techniques.
Problem Statement
Recruiters often receive hundreds of resumes for a single job position. Manually screening resumes is time-consuming and inefficient. This system automates resume parsing, classification, and job matching to streamline recruitment.
Key Features
• Resume parsing and skill extraction
• Machine Learning-based job classification
• Resume-to-job matching score calculation
• Admin and user interface
• Structured dataset processing
• Modular and scalable architecture
Technology Stack
• Python 3.10
• Flask (Web Framework)
• Scikit-learn (Machine Learning)
• Pandas & NumPy (Data Processing)
• HTML, CSS (Frontend)
• Pickle (.pkl) model deployment
Project Structure
app.py – Main Flask application
classifier.py – ML classification logic
matcher.py – Resume-job matching logic
resume_parser.py – Resume parsing module
templates/ – HTML templates
static/ – CSS and static files
dataset/ – Training and testing data
docs/ – Reports and documentation
requirements.txt – Required dependencies
